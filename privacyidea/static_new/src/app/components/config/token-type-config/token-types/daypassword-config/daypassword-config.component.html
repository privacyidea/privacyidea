DayPassword Token Settings

The DayPassword Token is a time based password token. It is based on the TOTP token, but it can be used more than once.

Here you can define settings, that will be set as default values, when enrolling a DayPassword token. Note that the user/admin policies daypassword_hashlib and daypassword_timestep always have precedence over these default values.
