SMS

The SMS Token is an event based token.

After the user has tried to authenticate with the OTP PIN, an SMS with an OTP value is sent to the users mobile phone. Then user can authenticate with this OTP value in a second step.

Here you can define how the SMS will be sent - via which kind of gateway.

SMS Gateway Configuration None @for (gateway of smsGateways(); track gateway) { {{ gateway }} } Select a predefined SMS Gateway configuration. OTP Validity Time The time in seconds for which the sent OTP value is valid for authentication.
